# Inventory Management System

An Inventory Management System built with the following technologies:
 - A fast Node.js RESTful Api backend
 - A Robust UI built with React

#Description One
An inventory management system (or inventory system) is the process by which
you track your goods throughout your entire supply chain, from purchasing to
production to end sales. It governs how you approach inventory management for your business.

# Description Two
Inventory management software is a software system for tracking
inventory levels, orders, sales and deliveries. It can also be used in
the manufacturing industry to create a work order, bill of materials
and other production-related documents.
